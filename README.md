# 写真台帳自動作成ツール (Photo Ledger System)

## 概要

大量の画像（現場写真や画面キャプチャなど）を読み込み、指定のフォーマットでWord文書に自動転記するVBAマクロツールです。

業務PC環境下でも確実に動作するよう、外部プログラム（Python等）への依存をなくし、Microsoft Wordネイティブの **VBA（マクロ）** を採用して開発しています。

---

## 主な機能

### 完全自動レイアウト

* 画像を最大3枚ずつのグループに分割
* 自動で表を作成

  * 左列：画像
  * 右列：ファイル名

### 自動見出し生成

* 選択したフォルダの階層構造（サブフォルダ名）を読み取り
* Word標準の「見出し」スタイルを自動適用

### 余白の最適化

* ページ余白を **「狭い（上下左右 1.27cm）」** に自動設定
* 限られたページスペースを有効活用

### 画像サイズの最適化

* アスペクト比（縦横比）を維持
* 表のセル幅に合わせて自動縮小

---

## フォルダ構成と見出し生成ルール

本ツールは、選択したフォルダ内の階層構造をもとに見出しレベルを自動生成します。

### フォルダ構成例

```text
📁 選択する大元フォルダ（例：社内システム_テストエビデンス）
├─ 📁 01_ログイン機能              ← 見出し1
│  ├─ 📁 正常系テスト             ← 見出し2
│  │  ├─ 📄 login_success.png
│  │  └─ 📄 redirect_top.png
│  └─ 📁 異常系テスト             ← 見出し2
│     └─ 📄 error_message.png
└─ 📁 02_ユーザー管理機能          ← 見出し1
   └─ 📄 user_list_view.png
```

### 出力イメージ

```text
見出し1：01_ログイン機能

  見出し2：正常系テスト
    └ 画像一覧表

  見出し2：異常系テスト
    └ 画像一覧表

見出し1：02_ユーザー管理機能
  └ 画像一覧表
```

## 導入手順（セットアップ）

### 1. VBAコードをコピー
本リポジトリの `PhotoLedger.bas` を開き、記載されているコードをすべてコピーします。

### 2. Wordを起動
ひな形となる文書、または新規の白紙文書を開きます。

### 3. VBAエディタを起動
キーボードの `Alt + F11` を押してVBAエディタを開きます。

### 4. 保存先（テンプレート）の選択
左側のプロジェクトウィンドウから **`Normal`** をクリックして選択します。
*(※これを行わないと、現在開いている文書のみに保存されてしまいます)*

### 5. コードを貼り付け
画面上部のメニューから「挿入」＞「標準モジュール」をクリックし、白紙の画面にコードを貼り付けます。

### 6. 保存して完了
`Ctrl + S` を押して設定を保存し、VBAエディタを閉じます。

## 使い方

### 1. Word文書を開く

マクロを導入したWord文書を開きます。

### 2. マクロ一覧を表示

以下のいずれかを実行します。

* 「表示」タブ → 「マクロ」
* `Alt + F8`

### 3. マクロを実行

一覧から

```text
CreatePhotoLedger
```

を選択して実行します。

### 4. フォルダを選択

画像が格納されている対象フォルダを選択し、「OK」を押します。

### 5. 完了を待つ

処理が終了すると

```text
写真台帳の作成が完了しました！
```

というメッセージが表示されます。

> ※処理中はWordの画面更新を停止しています。

## 対応ファイル形式

以下の画像形式に対応しています。

* `.jpg`
* `.jpeg`
* `.png`
* `.gif`
* `.bmp`

---

## 技術仕様

* Microsoft Word VBAのみで動作
* Python等の外部実行環境は不要
* フォルダ階層から見出しレベルを自動生成
* 画像サイズを自動調整
* 3枚1組の写真台帳フォーマットを自動作成

---

## 備考

* 既存のWordテンプレート（`Normal.dotm` など）に設定されている見出しスタイルをそのまま利用します。
* セキュリティ制約の強い業務PC環境での利用を想定しています。
* Word単体で動作するため、追加ソフトウェアのインストールは不要です。
